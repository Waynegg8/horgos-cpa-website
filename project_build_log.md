# 霍爾果斯會計師事務所網站構建日誌

## 項目概述
為霍爾果斯會計師事務所建立專業的企業網站，提供會計、稅務、審計等服務的線上展示平台。

## 技術架構
- **靜態網站生成器**：Hugo
- **前端框架**：原生 HTML/CSS/JavaScript
- **樣式預處理器**：SCSS
- **後端服務**：Cloudflare Workers
- **數據存儲**：Cloudflare KV
- **部署平台**：Cloudflare Pages
